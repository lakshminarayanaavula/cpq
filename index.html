Quick Start

  1. Enter your JWT token in the sidebar and click "Load All Data"
  2. Create Product Families to organize your catalog
  3. Add Product Lines within each family
  4. Create Products (Standard or Bundle type)
  5. Define Global Attributes for product configuration
  6. For Bundles: Add Features and Feature Products
  7. Create Constraint Rules for business logic
  8. Test your configuration in Evaluate Config
